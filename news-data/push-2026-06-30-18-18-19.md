### 1️⃣ Anthropic Labs 负责人：正与 SpaceX 洽谈，AI 未来将走向“结果定价”

* **核心亮点**：Anthropic Labs 负责人、Instagram 联合创始人 Mike Krieger 在最新峰会中透露，Anthropic 正在与 SpaceX 进行太空领域对话，并首次提出行业可能从“按 Token 计费”转向“基于结果的定价模式”。
* **SpaceX 合作**：Krieger 确认与 SpaceX 正在聊与空间、太空相关的事情，并表示自己从业内人士处了解到太空数据中心“无限电力+无限土地”的逻辑可信，尽管并非其专业领域。
* **结果定价尝试**：已推出的 Claude Managed Agents 提供“基于结果”模式——用户定义好结果和评估细则，模型执行完后按结果计费。如果市场接受，将重塑整个 API 经济。
* **AI 还只走了 10%**：Krieger 认为当前 AI 能力与终极形态之间只走了 10% 的路，接下来最重要的两个方向是：给 Claude 更多“自主权与环境感知”，以及弥合能力与实际应用之间的鸿沟。
* **案例佐证**：他用一个 100 万行级代码从 Python 转 TypeScript 的例子说明，Claude Code 已能完成大规模自主规划与执行，过去“管理 AI 像管理实习生”的说法已经过时。

### 2️⃣ ChatGPT 月活跃用户创历史新高：6.269 亿

* **核心亮点**：Similarweb 数据显示，ChatGPT 在 2026 年 5 月达到 6.269 亿月活跃用户，刷新历史纪录。
* **数据来源**：该数据由 Similarweb 官方账号在 X 平台披露，由 AI Will 账号转发确认。

### 3️⃣ Unitree 四步策略：在人形机器人市场尚未存在时培育需求

* **核心亮点**：一份基于 Unitree IPO 招股书和 UBTECH 公开文件的深度报告，系统拆解了中国公司在人形机器人市场如何通过“四级阶梯”打法，在没有需求时创造需求。
* **四级阶梯**：
  1. 科研销售为工具和设备成本提供资金
  2. 消费级/商用市场为规模化制造提供资金
  3. 政府采购为大规模机群部署提供资金
  4. 工业试点反哺真实世界训练数据
* **对比美国**：美国玩家（如 Figure、Tesla）跳过前三步直接冲刺工业市场，靠大额风投填补研发缺口。报告将中国路径称为“登梯” vs 美国路径的“冲刺”，并追问哪一条先到达工业规模。
* 🔗 [完整报告](https://corematter.substack.com/p/the-involuted-biped-the-humanoid)

### 4️⃣ Google 开源 design.md：让 AI 生成 UI 告别“Template 味”

* **核心亮点**：Google Labs 开源 design.md 项目，定义了一套让 AI 理解设计系统的标准化文件格式，GitHub 星数近 2 万。
* **文件结构**：YAML front matter 放机器可读的设计 token（颜色十六进制、字号、间距、圆角），Markdown 正文放人可读的设计理念。AI 既拿到精确值又理解意图。
* **配套 CLI**：`npx @google/design.md lint` 可自动校验 WCAG 对比度、缺失主色等，支持导出 Tailwind 主题配置。
* **生态跟进**：已有 VoltAgent 汇集 70+ 品牌样本库（awesome-design-md），以及第三方聚合站 designkit.sh。
* 🔗 [GitHub 仓库](https://github.com/google-labs-code/design.md)

### 5️⃣ 开源视频蒸馏工具 cangjie-skill 2.0 发布

* **核心亮点**：开源项目 cangjie-skill 升级至 2.0，支持从 YouTube、B 站、抖音等平台下载视频并蒸馏为可复用“技能包”，用于 Agent 问答。
* **技术栈**：使用美团 Longcat-2.0（1.6 万亿参数 MoE，国产芯片）结合 Claude Code 驱动工作流；视频转文字借用开源 ASR 和 yt-dlp。
* **实测效率**：蒸馏吴恩达 2026《AI 入门课》26 个视频（共 4 小时）耗时约 1 小时，生成 25 个技能包；蒸馏 80 分钟 loop engineering 视频耗时 22 分钟，生成 8 个技能包。
* **开发者受益**：可对长视频、播客等实时内容进行知识萃取，构建私有 skill 库，降低 Token 消耗、提升回答精度。
* 🔗 [开源地址](https://github.com/kangarooking/cangjie-skill)
